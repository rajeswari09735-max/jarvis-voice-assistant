# Jarvis Voice Assistant

A simple Python voice assistant built using speech recognition.

## Features
- Voice command recognition
- Play songs on YouTube
- Open applications like Notepad, Chrome, Calculator
- Tell current time
- Search information on Wikipedia

## Libraries Used
- pyttsx3
- SpeechRecognition
- pywhatkit
- wikipedia
- pyaudio

## Installation

Clone the repository:

```
git clone https://github.com/rajeswari09735-max/jarvis-voice-assistant.git
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the assistant:

```
python jarvis_voice_assistant_v2.py
```
